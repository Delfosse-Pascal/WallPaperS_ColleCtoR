# WallPaperS_ColleCtoR

Collectionneur de fonds d'écran.

## Description

Outil pour récupérer, organiser et gérer une collection de wallpapers.

## Fonctionnalités

- Téléchargement de wallpapers depuis sources multiples
- Organisation par catégories / résolutions / tags
- Prévisualisation
- Application directe en fond d'écran système

## Installation

```bash
git clone https://github.com/Delfosse-Pascal/WallPaperS_ColleCtoR.git
cd WallPaperS_ColleCtoR
```
